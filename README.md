This repository hosts work into modelling binary planetesimals embedded in the protoplanetary disk with the [Pencil Code](https://github.com/pencil-code/pencil-code). 

More documentation on the Pencil Code can be found at the following address: https://pencil-code.org/doc.php
Alternatively, the Pencil Code manual may be found at: https://pencil-code.readthedocs.io/en/latest/manuals/fmanual-index.html
